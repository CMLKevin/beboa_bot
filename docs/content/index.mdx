# Welcome to Beboa

Beboa is an AI-powered Discord companion bot designed for the BubbleBebe community. She's a magical talking snake - actually a cat in a snake costume who insists she's real - with a dynamic personality that evolves through interactions.

## What Makes Beboa Special

Unlike traditional Discord bots, Beboa features a sophisticated AI system that creates genuine, evolving relationships with users:

- **Dynamic Personality** - 14 personality traits that shift based on interactions
- **Mood System** - 11 moods that affect how she responds in real-time
- **Semantic Memory** - Long-term memory using vector embeddings
- **Relationship Tracking** - Individual relationships with each user
- **Natural Language Admin** - Jarvis Mode 2.0 for intuitive server management

## Core Features

### Engagement System

The Bebits economy keeps your community active:

| Feature | Description |
|---------|-------------|
| Daily Check-ins | Earn 1 Bebit per day |
| Streak Tracking | 72-hour grace period before reset |
| Leaderboard | Competitive community rankings |
| Reward Shop | 11 tiers from 1-500 Bebits |

### AI Chat

Mention @Beboa anywhere to start a conversation. She maintains context across messages and remembers important details about each user.

### Jarvis Mode 2.0

Server admins can use natural language to manage the server:

```
"give kevin 50 bebits"
"reset sarah's streak"
"show me the leaderboard"
```
