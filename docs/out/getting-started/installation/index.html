Getting Started

Installation

Step-by-step guide to installing Beboa on your system.

Installation

This guide walks you through setting up Beboa from scratch.

Prerequisites

  • Node.js 20+ - Download
  • Discord Bot Application - Developer Portal
  • OpenRouter API Key - Get one here

Step 1: Clone the Repository

bash
git clone https://github.com/CMLKevin/beboa_evo.git
cd beboa_evo

Step 2: Install Dependencies

bash
npm install

This installs:

  • discord.js - Discord API wrapper
  • better-sqlite3 - SQLite database
  • dotenv - Environment variable management

Step 3: Create Discord Bot Application

  1. Go to Discord Developer Portal
  2. Click New Application
  3. Name it "Beboa" (or your preferred name)
  4. Go to Bot section
  5. Click Reset Token and copy it (you'll need this)
  6. Enable these Privileged Gateway Intents:
    • ‚úÖ Message Content Intent
    • ‚úÖ Server Members Intent (optional)

Step 4: Get IDs from Discord

Enable Developer Mode in Discord:

  • User Settings ‚Üí Advanced ‚Üí Developer Mode

Then right-click to copy IDs:

IDHow to Get
CLIENT_IDDeveloper Portal ‚Üí General Information ‚Üí Application ID
GUILD_IDRight-click your server ‚Üí Copy Server ID
CHECKIN_CHANNEL_IDRight-click the check-in channel ‚Üí Copy Channel ID
NOTIFICATION_CHANNEL_IDRight-click notification channel ‚Üí Copy Channel ID
ADMIN_ROLE_IDServer Settings ‚Üí Roles ‚Üí Right-click role ‚Üí Copy Role ID
BEBE_USER_IDRight-click user ‚Üí Copy User ID

Step 5: Configure Environment

bash
cp .env.example .env

Edit .env with your values - see Configuration for all options.

Step 6: Invite Bot to Server

Generate invite URL in Developer Portal:

  1. Go to OAuth2 ‚Üí URL Generator
  2. Select scopes: bot, applications.commands
  3. Select permissions:
    • Send Messages
    • Embed Links
    • Read Message History
    • Use Slash Commands
    • Add Reactions
  4. Copy and open the generated URL

Step 7: Start the Bot

bash
npm start

You should see:

[DATABASE] Connected to data/beboa.db
[BOT] Logged in as Beboa#1234
[BOT] Ready! Serving 1 guild

Troubleshooting

"Missing required environment variables"

Ensure all variables in .env are set. See Configuration.

"Invalid token"

Regenerate your bot token in the Developer Portal.

"Missing Access" errors

Check bot permissions and ensure it's invited with correct scopes.

Database errors

Delete data/beboa.db to reset (you'll lose data).
