Beboa

A Discord bot with personality

Beboa is an AI-powered Discord companion that remembers your conversations, evolves her personality through interactions, and manages your community with natural language commands. Bratty, snarky, and secretly caring.

Get StartedView on GitHub

Core Features

Dynamic Personality

14 evolving traits, 11 moods, and per-user relationship tracking that creates genuine connections.

Semantic Memory

User-specific memories and server-wide awareness using vector embeddings for true context recall.

Jarvis Mode

Natural language admin commands with LLM-powered intent parsing. "Give Kevin 100 bebits" just works.

Engagement System

Daily check-ins, Bebits currency, streak tracking, and an 11-tier reward shop.

Quick Start

1

Clone Repository

git clone https://github.com/CMLKevin/beboa_evo.git
2

Install Dependencies

cd beboa_evo && npm install
3

Configure Environment

cp .env.example .env
4

Start the Bot

npm start
